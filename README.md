Automatic Building Maker


What does it do?
>creates a building with fully random attributes by deafult
>attributes for each element can be edited as desired by the user

### Functions:

### fortress_geometry.py
- `create_floor(floor_length, floor_thickness, floor_length, position)` -a rectangular floor
- `create_wall(wall_width, wall_height, wall_thickness, position)` — a rectangular wall
- `create_roof(roof_width, roof_height, roof_length, position)` — a rectangular roof
